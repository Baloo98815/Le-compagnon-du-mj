🔌 INTÉGRATION API
═════════════════════════════════════════════════════════════════════════════

Endpoints utilisés:

campaignsAPI:
  • getAll() - Récupère toutes les campagnes
  • getById(id) - Récupère une campagne spécifique
  • create(data) - Crée une nouvelle campagne
  • update(id, data) - Met à jour une campagne
  • delete(id) - Supprime une campagne
  • addPlayer(campaignId, playerId) - Ajoute un joueur
  • removePlayer(campaignId, playerId) - Retire un joueur

playersAPI:
  • getAll() - Récupère tous les personnages
  • getById(id) - Récupère un personnage spécifique
  • create(data) - Crée un nouveau personnage
  • update(id, data) - Met à jour un personnage
  • delete(id) - Supprime un personnage
  • uploadToken(id, file) - Upload la photo du token

scenesAPI:
  • getAll(campaignId) - Récupère les scènes d'une campagne
  • create(data) - Crée une nouvelle scène
  • delete(id) - Supprime une scène
